# Alat Pembaca Bilangan

## Cara menggunakan

- Jalankan ` npm run preview`
- Buka port yang terbuka
- Tes halaman web
- Cek apakah teks benar dengan bilangan yang diinput

> Beritahu kesalahan teks output yang tampil di issues

---

Untuk menggunakan alat pembaca ini di project anda, salin file `pembacaAngka.js` ke folder project, kemudian import module `convertNumberToText`, lalu jalankan dengan sintaks `convertNumberToText(bilanganAsli, desimal)`

### Pemberitahuan

- Kedua parameter harus string
- Tidak bisa untuk bilangan negatif
- Hanya bahasa Indonesia
